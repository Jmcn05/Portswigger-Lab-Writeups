# PortSwigger Web Security Academy Lab Writeups

This repository contains my personal writeups for PortSwigger Web Security Academy labs. These writeups document my learning process, methodology, and understanding of common web application vulnerabilities.

## Purpose

The purpose of this repository is to demonstrate practical knowledge of web application security concepts in a legal, controlled lab environment.

## Topics Covered

- SQL Injection
- Cross-Site Scripting
- Cross-Site Request Forgery
- Server-Side Request Forgery
- Authentication Vulnerabilities
- Access Control Vulnerabilities
- File Upload Vulnerabilities
- Business Logic Vulnerabilities
- XML External Entity Injection
- JWT Attacks
- OAuth Vulnerabilities
- Web Cache Poisoning

## Writeup Format

Each lab writeup generally includes:

- Lab objective
- Vulnerability summary
- Step-by-step exploitation process
- Explanation of why the vulnerability works
- Remediation or prevention notes
- Key takeaways

## Disclaimer

All writeups in this repository are for educational purposes only. The techniques demonstrated are performed exclusively in authorized lab environments provided by PortSwigger Web Security Academy. I do not condone or support unauthorized testing against real systems. As of making this repository, not all topics listed have been covered. The repository will be updated as writeups are made for the labs.
